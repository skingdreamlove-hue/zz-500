# 中证500量化系统 - 桌面应用使用说明

## 🚀 应用简介

中证500量化系统是一款专业的跨平台桌面应用，集成了实时市场数据监测、智能网格交易助手、风险控制预警等功能。采用Electron框架开发，支持Windows、macOS和Linux三大主流操作系统。

## 📦 安装指南

### Windows用户
1. 下载 `中证500量化系统 Setup 1.0.0.exe`
2. 双击运行安装程序
3. 选择安装路径（默认：`C:\\Program Files\\中证500量化系统`）
4. 勾选"创建桌面快捷方式"
5. 点击"安装"完成安装

### macOS用户
1. 下载 `中证500量化系统-1.0.0.dmg`
2. 双击打开DMG文件
3. 将应用图标拖拽到"Applications"文件夹
4. 在Launchpad或应用程序文件夹中启动应用

### Linux用户
1. 下载 `中证500量化系统-1.0.0.AppImage`
2. 给文件添加执行权限：`chmod +x 中证500量化系统-1.0.0.AppImage`
3. 双击运行或通过终端启动：`./中证500量化系统-1.0.0.AppImage`

## 🎯 首次使用

### 启动应用
- **Windows**: 双击桌面快捷方式或开始菜单中的"中证500量化系统"
- **macOS**: 在Launchpad或应用程序文件夹中点击应用图标
- **Linux**: 双击AppImage文件或通过终端运行

### 初始化过程
1. 应用启动后会自动检测并启动后端服务
2. 等待状态栏显示"✅ 后端服务运行中"
3. 系统自动获取最新市场数据
4. 界面加载完成后即可正常使用

## 📊 主要功能

### 1. 实时市场监测
- **中证500指数**: 实时价格和涨跌幅
- **全市场成交额**: 双数据源交叉验证
- **换手率分析**: 历史分位值显示
- **北向资金**: 20日累计净流入/出

### 2. 估值与趋势分析
- **ERP分位**: 股权风险溢价分析
- **MA250偏离**: 长期趋势判断
- **MA60趋势**: 中期趋势方向
- **市场广度**: 5日涨跌比例

### 3. 网格交易助手
- **智能网格计算**: 自动计算买卖价位
- **ETF换算**: 支持510500 ETF交易
- **持仓保护**: 浮亏预警和止损建议
- **信号提示**: 买卖时机提醒

### 4. 风险控制
- **止盈预警**: 4项预警指标监控
- **仓位管理**: 智能仓位建议
- **趋势折扣**: MA60趋势影响仓位上限

## 🔧 操作指南

### 数据刷新
- 点击顶部"🔄 刷新数据"按钮
- 使用快捷键 `Ctrl+R` (Windows/Linux) 或 `Cmd+R` (macOS)
- 数据每30秒自动更新一次

### 网格交易设置
1. 设置总测试资金（默认10000元）
2. 输入网格基准价（建议使用MA250）
3. 设置网格步长（系统推荐3%）
4. 输入ETF参考价格
5. 点击"计算网格档位"

### 持仓保护
1. 输入已投入金额
2. 输入建仓均价
3. 系统自动计算浮亏比例
4. 根据预警级别调整策略

## 🔄 更新与维护

### 自动更新
- 应用启动时自动检查新版本
- 支持后台下载和静默安装

## 🛠️ 故障排除

### 常见问题

#### 应用无法启动
1. 检查系统是否满足最低要求
2. 尝试以管理员权限运行
3. 重新安装应用

#### 数据获取失败
1. 检查网络连接是否正常
2. 确认防火墙未阻止应用
3. 重启应用尝试重新连接

#### 后端服务异常
1. 应用会自动尝试重启服务
2. 如持续失败，请重启应用
3. 检查系统日志获取详细信息

### 系统要求

#### 最低配置
- **操作系统**: Windows 10 / macOS 10.14 / Ubuntu 18.04+
- **内存**: 4GB RAM
- **存储**: 500MB可用空间
- **网络**: 宽带互联网连接

#### 推荐配置
- **操作系统**: Windows 11 / macOS 12+ / Ubuntu 20.04+
- **内存**: 8GB RAM或更多
- **存储**: 1GB可用空间
- **网络**: 稳定的宽带连接
